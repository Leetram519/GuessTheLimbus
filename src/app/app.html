How to Play

🎯 Objective: Guess today's Limbus Company Identity in 4 tries or less.

📅 Daily Challenge: A new ID is selected each day at midnight.

🔍 How to Guess

  • Use the search bar to filter available IDs
  • Click on an ID card to select it
  • Confirm your choice in the popup
  • Analyze the comparison results to narrow down your next guess

✓ Understanding the Feedback

After each guess, you'll see a Quick Summary and Detailed Comparison:

Color Guide:

  • 🟢 Green / YES: The attribute matches the target ID perfectly
  • 🔴 Red / NO: The attribute doesn't match the target ID
  • 🟠 Orange / ~ / LESS/MORE: Partial match or directional hint

Statistics Explained:

  • Sinner: Which of the 12 Sinners this ID belongs to (Yi Sang, Faust, Don Quixote, etc.)
  • Rarity: The tier of the ID
  • Season: When the ID was released (Standard Fare, 1 = Season 1, 2 = Season 2, Walpurgisnacht I, Walpurgisnacht II, etc.)
  • Passives: Total number of passive abilities (combat + support)
    • ↓ LESS: You need to guess an ID that has LESS passives than your current guess
    • ↑ MORE: You need to guess an ID that has MORE passives than your current guess
  • Precise Keywords: Faction or affiliation keywords (e.g., "Seven Association", "Molar Office", "Fixer")
    • Each keyword from YOUR guess is checked individually
    • YES means that keyword is also on the target ID
    • NO means that keyword is not on the target ID
    • ~ in Quick Summary means some keywords match, but not all
  • Status Keywords: Status effects the ID inflicts (e.g., "Rupture", "Tremor", "Sinking")
    • Works the same as Precise Keywords
  • Skills: Each ID has 1-3 skills, and each skill may have variations
    • Sin Affinity: The sin type (Wrath, Lust, Sloth, Gluttony, Gloom, Pride, Envy)
    • Coin Count: Number of coin flips the skill uses (1-3)
    • Final Power: The maximum amount of clash power that you can do with this skill, excluding any passives or buffs. (For instance, Grip Sinclair's Skill 3 has 30 Final Power, and Thumb Meursault's Skill 3-2 has 18 Final Power)
      • ↓ LESS: You need to guess an ID that has LESS Final Power than your current guess
      • ↑ MORE: You need to guess an ID that has MORE Final Power than your current guess

💡 Strategy Tips

  • LESS/MORE hints for passives and skill power can help you triangulate the exact values
  • Try to aim for somewhat common IDs, such as the N Corp. IDs, or the Wuthering Heights IDs
  • IDs that do multiple status effects are really good to guess
  • Guessing Ring Sang is cheating

⚠️ Important Info

  • Ring Sang and Ring Outis have been nerfed (They only count as bleed). This is due to Kim Jihoon hating them both.

⏰ Reset: The game resets daily at midnight with a new ID to guess.
