How to Play

🎯 Objective

Guess today's Limbus Company Identity in 4 tries or less

📅 Daily Challenge

A new ID is selected each day at midnight (France timezone)

⏰ Reset Time

The game resets daily at midnight with a new ID to guess

🔍 How to Guess

  • Use the search bar to filter available IDs
  • Click on an ID card to select it
  • Confirm your choice in the popup
  • Analyze the comparison results to narrow down your next guess

✓ Understanding the Feedback

After each guess, you'll see a Quick Summary and Detailed Comparison:

Color Guide:

  • 🟢 Green / YES: The attribute matches the target ID perfectly
  • 🔴 Red / NO: The attribute doesn't match the target ID
  • 🟠 Orange / ~ / LESS/MORE: Partial match or directional hint

📊 Statistics Explained

Sinner: Which of the 12 Sinners this ID belongs to (Yi Sang, Faust, Don Quixote, etc.)
Rarity: The tier of the ID (0 = base, 00 = 2-star, 000 = 3-star)
Season: When the ID was released (Standard Fare, Season 1, Season 2, Walpurgisnacht I, II, etc.)
Passives: Total number of passive abilities (combat + support)
  • ↓ LESS: Target ID has fewer passives than your guess
  • ↑ MORE: Target ID has more passives than your guess
Precise Keywords: Faction or affiliation keywords (e.g., "Seven Association", "Molar Office", "Fixer")
  • Each keyword from YOUR guess is checked individually
  • YES = Keyword is also on the target ID
  • NO = Keyword is NOT on the target ID
  • ~ (in summary) = Some keywords match, but not all
Status Keywords: Status effects the ID inflicts (e.g., "Rupture", "Tremor", "Sinking")
  • Works the same as Precise Keywords - each is checked individually
Skills: Each ID has 1-3 skills, and each skill may have variations
  • Sin Affinity: The sin type (Wrath, Lust, Sloth, Gluttony, Gloom, Pride, Envy)
  • Coin Count: Number of coin flips the skill uses (1-3)
  • Final Power: Maximum clash power from this skill (excluding passives/buffs)
    • ↓ LESS: Target ID has lower Final Power
    • ↑ MORE: Target ID has higher Final Power

💡 Strategy Tips

💪 Use LESS/MORE Hints: These directional hints for passives and skill power help you triangulate the exact values
🎯 Smart Guessing: Try common IDs like N Corp., Wuthering Heights, or IDs with multiple status effects
🔑 Keywords are Gold: IDs with multiple unique keywords can quickly narrow down possibilities

⚠️ Important Info

Nerfed IDs: Ring Sang and Ring Outis only count as "Bleed" (Kim Jihoon hates them both)
